# 日記區塊模板 - 參考小布的格式

## 常用區塊標題（按 emoji 分類）

### 📋 今日重要事項
- 📋 今日重大突破（一次性盤點）
- 📋 今日完成
- 📋 今日學習

### 📍 書籤相關
- 📍 書籤新增（日期）
- 📍 收藏推薦

### ⛽ 時間廣播
- ⛽ 早上廣播（月份 日期）
- ⛽ 晚間廣播

### 💭 思考/反思
- 💭 小小心得
- 💭 反思
- 💭 觀察

### 🔧 技術記錄
- 🔧 Bug 解決方案
- 🔧 技術筆記

### 🌐 資訊整理
- 🌐 重要資訊
- 🌐 產業動態

### 📊 資料
- 📊 市場資料
- 📊 統計

### 🎯 計畫
- 🎯 明日計畫

## 使用建議
- 不需要每天都有所有區塊
- 根據當天內容選擇 1-3 個重點
- 區塊標題 + 內容 = 清晰的日記結構
